# **App Name**: Project Nexus

## Core Features:

- Daily Pulse: Aggregate real-time campus information such as dining, events, and alerts, summarized for quick viewing.
- Personalized Schedule: Display a clear, personalized view of the user's class schedule, incorporating real-time updates and notifications of changes.
- Student Exchange Marketplace: Facilitate a platform for students to exchange goods and services such as textbooks and electronics.
- Campus Pathfinder: Provide interactive campus maps with AR navigation, accessibility routes, and discovery of campus buildings and hidden spots.
- AI-Powered Insights Tool: Uses sentiment and behavior analysis tools for mood tracking and potential personalization of campus events/resources. Functions as a tool to support matching campus resources to student needs.
- Automated Summaries: Summarize long form documents into bulleted, succinct takeaways

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to represent intelligence, focus, and a connection to academia.
- Background color: Light lavender (#F3E5F5), offering a soft and calming backdrop.
- Accent color: Fuchsia (#E91E63), providing a vibrant and modern contrast for key interactions.
- Body text: 'PT Sans', sans-serif, is appropriate for long lines of text
- Headline Font: 'Playfair', a modern sans-serif, provides an elegant, fashionable feel.
- Use clean, minimalist icons with a consistent style to represent different campus resources and categories.
- Emphasize a card-based layout for presenting information in a digestible format. The layout should also prioritize quick access to important functions and information.
